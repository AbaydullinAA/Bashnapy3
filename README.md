## 1. Наименование проекта

**WatchMate** – система рекомендации наручных часов, использующая гибридный подход: коллаборативную фильтрацию на основе оценок пользователей и контентную фильтрацию по характеристикам товара.

## 2. Описание проекта

Данный проект реализует два подхода к рекомендациям:

- **Коллаборативная фильтрация** – на основе синтетических оценок пользователей строится матрица предпочтений, вычисляется косинусное сходство между пользователями, и рекомендуются часы, которые высоко оценили похожие пользователи.
- **Контентная фильтрация** – часы сравниваются по нормализованным числовым признакам (цена, количество отзывов, рейтинг) и категориальным признакам (бренд, название) с использованием комбинации евклидова расстояния и коэффициента Жаккара.

Все расчёты выполнены на Python с применением библиотек `pandas`, `numpy`, `scikit-learn`, `matplotlib` и `seaborn`. Для интерактивного взаимодействия реализован Telegram‑бот, который позволяет пользователям оценивать часы, получать персональные рекомендации и визуализировать сходство с разными типами пользователей.

Проект демонстрирует:

- Предобработку данных (нормализацию, создание признаков)
- Реализацию кастомных метрик сходства
- Построение тепловой карты матрицы сходства
- Использование коллаборативной фильтрации с центрированием оценок
- Интеграцию с Telegram Bot API
- Нечёткий поиск (расстояние Левенштейна) для удобного ввода названий

## 4. Установка и запуск

### Требования
- Python 3.8 или выше
- Git (для клонирования репозитория)
- Аккаунт Telegram и токен бота (получить у [BotFather](https://t.me/BotFather))

### Шаги

1. **Клонировать репозиторий**
   ```bash
   git clone https://github.com/yourusername/watchmate.git
   cd watchmate
   ```

2. **Создать виртуальное окружение (рекомендуется)**
   ```bash
   python -m venv venv
   source venv/bin/activate       # Linux/macOS
   venv\Scripts\activate          # Windows
   ```

3. **Установить зависимости**
   ```bash
   pip install -r requirements.txt
   ```
   Содержимое `requirements.txt` (пример):
   ```
   pandas
   numpy
   scikit-learn
   matplotlib
   seaborn
   python-telegram-bot
   ```

4. **Подготовить данные**
   - Поместите файлы `watches_processed.csv` и `synthetic_ratings.csv` в корневую папку проекта.
   - Убедитесь, что в `watches_processed.csv` есть столбцы: `watch_name`, `brand_name`, `price`, `number_of_reviews`, `rating`. При необходимости добавьте `image_url` для отображения фотографий в боте (если нет – будет использоваться заглушка).

5. **Настроить токен бота**
   - В файле `bot.py` замените строку `TOKEN = "..."` на ваш токен, полученный от BotFather.

6. **Запустить бота**
   ```bash
   python bot.py
   ```
   Бот начнёт опрашивать сервер Telegram и отвечать на сообщения.

## 5. Использование

### Через Telegram‑бота

1. Найдите в Telegram вашего бота и отправьте команду `/start`.
2. Выберите режим:
   - **Коллаборативная рекомендация** – вам будет предложено оценить несколько часов от 1 до 5. Можно вводить названия с опечатками – бот предложит варианты. После ввода `/done` бот покажет 5 рекомендаций и построит график сходства с разными типами пользователей.
   - **Контентная рекомендация** – введите название часов (или его часть), бот покажет фото (если доступно) и список из 5 похожих моделей с коэффициентами сходства.

### Через командную строку (для разработки)

Файл `main_recommend.py` содержит те же алгоритмы и может использоваться для отладки или расчётов без бота:

```bash
python main_recommend.py
```

Затем выберите режим (1 – коллаборативный, 2 – контентный) и следуйте инструкциям.
